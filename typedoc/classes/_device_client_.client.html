Class Client

IoT Hub device client used to connect a device with an Azure IoT hub.

Users of the SDK should call one of the factory methods, {@link azure-iot-device.Client.fromConnectionString|fromConnectionString} or {@link azure-iot-device.Client.fromSharedAccessSignature|fromSharedAccessSignature} to create an IoT Hub device client.
